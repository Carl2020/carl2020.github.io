Summary: Latitude vs. X

Visualizations

This project's purpose was to show what aspects (e.g. maximum temperatures, wind speed, etc.)of the weather changes as we approach the equator. To do this, we first pulled data using the OpenWeatherMap API and assembled a dataset from over 500 randomly selected cities from around the world. After assembling the dataset, we used Matplotlib to plot the various features of weather vs. latitude. The features included were: temperature, wind speed, cloudiness and humidity. This site also provides a table containing the dataset and the 4 different visualizations created as part of the analysis, as well as explanations and descriptions of the correlations.
